# Cinema Bot 🎬

Telegram-бот для поиска информации о фильмах и сериалах.

## Возможности

- Поиск фильмов и сериалов по названию
- Информация: постер, рейтинг, описание, актёры, режиссёр
- Ссылки для просмотра (VK Video, Rutube, Dzen, YouTube)
- История поиска и статистика

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и краткая справка |
| `/help` | Подробная справка по использованию |
| `/history` | История ваших поисков |
| `/stats` | Статистика: какие фильмы искали чаще всего |

## Использование

Просто отправьте название фильма текстом:

```
Venom
Остров собак
Мстители: война бесконечности
```

## Технологии

- **Python 3.11+**
- **aiogram 3.x** — асинхронный Telegram Bot API
- **aiohttp** — асинхронные HTTP-запросы
- **aiosqlite** — асинхронная работа с SQLite
- **Kinopoisk Unofficial API** — данные о фильмах

## Запуск

### Локально
```bash
Установка зависимостей
pip install -r requirements.txt

Настройка переменных окружения
cp .env.example .env
Отредактируйте .env, добавив токены

Запуск
python -m src.main
```
### Docker
```bash
docker-compose up -d
```
