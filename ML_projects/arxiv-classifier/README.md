# arXiv Paper Classifier

Мультилейбл-классификатор научных статей по категориям arXiv на дообученном
SciBERT. На вход — заголовок и аннотация, на выходе — все подходящие категории
(у статьи их может быть несколько) с вероятностями.

🔗 **Живое демо:** https://huggingface.co/spaces/artem-volosatov/arxiv-classifier

## Задача

Статье на arXiv часто соответствует несколько рубрик, поэтому задача поставлена
как мультилейбл-классификация по ~130 категориям. Классы сильно несбалансированы,
поэтому качество мерилось по micro- и macro-F1.

## Подход

- **Модель:** SciBERT (BERT-base, ~100M параметров, словарь и предобучение под
  научную лексику) с классификационной головой на 130 выходов.
- **Данные:** [TimSchopf/arxiv_categories](https://huggingface.co/datasets/TimSchopf/arxiv_categories),
  ~163k train / 20k val / 20k test. На вход — конкатенация заголовка и аннотации,
  окно 256 токенов.
- **Обучение:** A100, 10 эпох (~150 мин), batch 128, bf16, лосс — binary
  cross-entropy.
- **Подбор порога:** снижение порога отсечения с 0.5 до 0.25 дало бесплатный
  прирост обеих метрик.

## Результаты

| Метрика  | Значение |
|----------|----------|
| Micro-F1 | 0.71     |
| Macro-F1 | 0.55     |

Модель за 4 эпохи освоила частые классы (Micro-F1 ≈ 0.65 при Macro ≈ 0.30),
дальнейшие эпохи подтягивали редкие — Macro рос до 0.50 при почти неизменном
валидационном лоссе. Подробный разбор — в [report.pdf](./report.pdf).
